# 📊 Telco Customer Churn Prediction

A machine learning project to predict customer churn in a telecom company and help improve customer retention strategies.

## 🚀 Overview
Customer churn is a critical problem in the telecom industry. This project builds predictive models to identify customers likely to leave, enabling proactive business decisions.
---
## 📁 Dataset
- File: WA_Fn-UseC_-Telco-Customer-Churn.csv  
- Contains customer demographics, services, billing, and churn status
---
## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- XGBoost  
- Imbalanced-learn (SMOTE)  
- Gradio  
---
## 🔍 Workflow
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering & Encoding  
- Handling Class Imbalance (SMOTE)  
- Model Training  
- Model Evaluation  
---
## 🤖 Models Used
- Logistic Regression  
- Random Forest  
- XGBoost  
---
## 📈 Results
| Model               | Accuracy |
|--------------------|----------|
| Logistic Regression | 76.08%   |
| Random Forest       | 77.15%   |
| XGBoost             | 76.93%   |
---
## 💡 Key Insights
- Month-to-month contracts have highest churn  
- Customers without tech support or security churn more  
- Fibre optic users show higher churn  
- Long-term customers rarely churn  
---
## 🧠 Business Recommendations
- Promote long-term contracts  
- Improve fibre service quality  
- Offer bundled add-ons (security & support)  
- Focus on retaining new customers
